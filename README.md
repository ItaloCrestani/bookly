## 📖 Sobre o projeto

O **Bookly** é uma aplicação web desenvolvida para melhorar sua experiência de leitura.

Além de permitir explorar milhares de livros, a aplicação possibilita criar uma estante pessoal, organizar livros por status de leitura, acompanhar o progresso e manter tudo sincronizado na sua conta.

## ✨ Funcionalidades

- Explorar livros utilizando a Google Books API
- Pesquisa de livros por título
- Página completa com informações detalhadas do livro
- Organização da estante por status
  - Quero ler
  - Estou lendo
  - Já li
- Controle do progresso de leitura
- Autenticação de usuários
- Sincronização dos dados com Firebase
- Tema claro e escuro
- Interface totalmente responsiva
- Skeleton Loading durante carregamentos
- Tratamento de erros e páginas não encontradas
